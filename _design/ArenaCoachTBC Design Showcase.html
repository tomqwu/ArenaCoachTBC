Every recommendation traces the signals that contributed. Every probability emit is calibrated against synthetic ground truth.

Top 3 score contributors + comp tag + confidence. Shown by default; hidden in compact mode.

Top-scoring CC chain from the comp's catalog, ranked by expected value across opponent responses. Each step shown with its DR category.

Lit when the friendly's ability is ready. Reminders only — never bound to actions.

Observed via combat log. PvP trinket pulses red when up.

scene 05 / 09chain planning detail

Chain planning · the engine's plan, made legible

A chain is an ordered list of CC links. Each is gated by DR (within-chain accumulation + observed history) and caster cooldown. ExpectedProb = product of effective multipliers; the picked chain is the one that survives lookahead against the most likely opponent response.

Sap off-healer, kidney the target

Kidney Shotenemy rogue · primary
STUNDR 1.00 × (fresh)

The chain encodes the enemy comp's canonical setup. By recognizing it the engine can predict the next 2–3 CC events and emit forward-looking callouts.

Each step's effective multiplier is the minimum of observed DR (from DRTracker) and within-chain accumulation. A third stun on the same target would walk 1.00 → 0.50 → 0.25; a fourth would reject as DR_immune.

Caster availability is checked against CooldownTracker. A pending cooldown returns cd_pending and zeroes the chain's expected probability — the lookahead drops it from contention.

Final ranking comes from Lookahead:Score — the chain prob is weighted by the opponent's likely response (read from OpponentProfile): if they reliably trinket the first CC of this chain, the chain's EV drops below the alternative.

scene 08 / 09weakaura bridge

WeakAura bridge · own your display

Five copy-paste WA templates ship in docs/weakaura-pack.md. They consume the _G.ArenaCoachTBC getter surface. Build your own HUD: huge centered mode + target, side-stream of callouts, mini chain ticker, opponent-profile readout — your call.

Template 2 · Burst gate (Icon)
-- Trigger (Status): show icon when ready
function()
    if not _G.ArenaCoachTBC then return false end
    return _G.ArenaCoachTBC.IsBurstAllowed()
       and _G.ArenaCoachTBC.GetMode() == "KILL"
end

-- Custom Text
function()
    local api = _G.ArenaCoachTBC
    if not api then return "" end
    return string.format(
        "%s on %s",
        api.GetMode() or "",
        api.GetPrimaryTargetName()
            or api.GetPrimaryTargetClass()
            or "")
end
Template 5 · Comp readout (Text)
function()
    local api = _G.ArenaCoachTBC
    if not api then return "" end
    local comp = api.GetEnemyCompLabel() or "?"
    local conf = api.GetCompSpecConfirmed()
        and "|cff66ff66spec-confirmed|r"
        or  "|cffcccc99class-guessed|r"
    return comp .. " (" .. conf .. ")"
end
